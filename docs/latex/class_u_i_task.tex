\doxysubsection{Detailed Description}
UI task with web server, Web\+Socket, and serial interface. 

Features\+:
\begin{DoxyItemize}
\item Serial commands\+: v, p, s, z, h/?
\item Web server\+: serves static files from SPIFFS
\item Web\+Socket\+: real-\/time bidirectional communication
\item REST API\+: HTTP endpoints for status/commands 
\end{DoxyItemize}
