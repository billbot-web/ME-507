\doxysubsection{Detailed Description}
Free\+RTOS task for BNO055 9-\/DOF IMU sensor data acquisition and distribution. 

This header defines a high-\/performance IMU task that continuously reads orientation, angular velocity, and acceleration data from the Adafruit BNO055 sensor. The task operates in NDOF (Nine Degrees of Freedom) fusion mode, providing pre-\/filtered and calibrated sensor data to the rest of the system via thread-\/safe Share$<$\+T$>$ objects.

Key Features\+:
\begin{DoxyItemize}
\item Continuous data streaming mode for real-\/time telemetry
\item NDOF sensor fusion (accelerometer + gyroscope + magnetometer)
\item Configurable sampling rate with millisecond precision
\item Built-\/in calibration status monitoring and persistence
\item FSM-\/based architecture for clean state management
\item Thread-\/safe data distribution to multiple consumers
\end{DoxyItemize}

Data Outputs\+:
\begin{DoxyItemize}
\item Euler Angles\+: Roll, pitch, yaw in degrees (absolute orientation)
\item Gyroscope\+: Angular velocity in degrees/second

\end{DoxyItemize}
