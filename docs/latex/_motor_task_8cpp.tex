\doxysubsection{Detailed Description}
Implementation of motor control task with velocity, position, and camera-\/based control modes. 

This file implements a comprehensive motor control system with multiple control modes\+:
\begin{DoxyItemize}
\item WAIT\+: Idle state with motor braked
\item VRUN\+: Velocity control with feedforward and stiction compensation
\item PRUN\+: Position control using PID with anti-\/windup
\item CAMERA\+\_\+\+PRUN\+: Camera error-\/based position control for visual tracking
\end{DoxyItemize}

Features include\+:
\begin{DoxyItemize}
\item Independent PID controllers for velocity and position control
\item Tilt motor position clamping for safety (-\/150° to +10°)
\item Stiction compensation to overcome static friction
\item Dynamic state transitions based on command shares
\end{DoxyItemize}
