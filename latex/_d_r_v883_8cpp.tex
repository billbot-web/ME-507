\doxysubsection{Detailed Description}
Implementation of \doxylink{class_d_r_v883}{DRV883} motor driver class for ESP32-\/based motor control. 

This file provides the complete implementation of the \doxylink{class_d_r_v883}{DRV883} motor driver interface, utilizing the ESP32\textquotesingle{}s hardware LEDC (LED Control) peripheral for high-\/frequency PWM generation. The implementation supports bidirectional motor control with configurable PWM parameters.
