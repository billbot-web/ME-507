The \doxylink{class_f_s_m}{FSM} does not take ownership; ensure the pointed-\/to States outlive the \doxylink{class_f_s_m}{FSM}.

Execute the current state\textquotesingle{}s behavior (initializing to the first entry on startup).

Selection rule\+:
\begin{DoxyItemize}
\item This \doxylink{class_f_s_m}{FSM} is execute-\/driven (no predicate-\/based scanning).
\item On creation it is initialized with a state table; the first update will initialize the current state to the first entry in that table.
\item Thereafter \doxylink{class_f_s_m_a593d3fc27483578741d0e07a341b2383}{run\+\_\+curstate()} calls the current state\textquotesingle{}s execute() and honors any returned next-\/state ID to perform a forced transition.
\end{DoxyItemize}
