\doxysubsection{Detailed Description}
Simple dual-\/pin PWM motor driver class for \doxylink{class_d_r_v883}{DRV883} series. 

Controls motor direction using IN1/\+IN2 pins and speed using ESP32 LEDC PWM. Effort range\+: -\/100 to 100 (\%)
\begin{DoxyItemize}
\item Positive = forward (PWM on IN1)
\item Negative = reverse (PWM on IN2)
\item Zero = motor stopped (both low) 
\end{DoxyItemize}
