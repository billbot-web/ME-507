\doxysubsection{Detailed Description}
Table-\/driven state descriptor with execution behavior. 

A \doxylink{class_state}{State} is defined by\+:
\begin{DoxyItemize}
\item a small integer ID
\item a const char\texorpdfstring{$\ast$}{*} name (for debugging)
\item an execute function that implements the state\textquotesingle{}s behavior and may return the next state ID to transition to
\end{DoxyItemize}

The \doxylink{class_f_s_m}{FSM} is execute-\/driven\+: it calls the current state\textquotesingle{}s execute() and honors any returned next-\/state ID to perform forced transitions.
