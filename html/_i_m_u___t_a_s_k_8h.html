Detailed Description

FreeRTOS task for BNO055 9-DOF IMU sensor data acquisition and distribution.

This header defines a high-performance IMU task that continuously reads orientation, angular velocity, and acceleration data from the Adafruit BNO055 sensor. The task operates in NDOF (Nine Degrees of Freedom) fusion mode, providing pre-filtered and calibrated sensor data to the rest of the system via thread-safe Share<T> objects.

Key Features:

  • Continuous data streaming mode for real-time telemetry
  • NDOF sensor fusion (accelerometer + gyroscope + magnetometer)
  • Configurable sampling rate with millisecond precision
  • Built-in calibration status monitoring and persistence
  • FSM-based architecture for clean state management
  • Thread-safe data distribution to multiple consumers

Data Outputs:

  • Euler Angles: Roll, pitch, yaw in degrees (absolute orientation)
  • Gyroscope: Angular velocity in degrees/second
  • Accelerometer: Linear acceleration in m/sÂ²
